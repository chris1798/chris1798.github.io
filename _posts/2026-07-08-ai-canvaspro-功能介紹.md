## 什麼是 AI Canvas Pro？

[AI Canvas Pro](https://github.com/ashuoAI/AI-CanvasPro)（作者：阿碩）是一款基於節點的 AI 多模態畫布編輯器，以 Electron 桌面應用形式發布。它讓你可以在無限畫布上，透過可視化節點連線的方式，自由組合 AI 能力，生成文本、圖像、影片與音訊。

目前已支持接入 RH AI 應用、ComfyUI 雲端工作流和 ComfyUI 本地工作流，將外部模型與工作流能力統一帶入畫布節點中編排複用。

> ⚠️ **授權聲明**：本項目為「公開發布」的 Electron 桌面應用，並非開源軟體。未經作者书面授权，禁止用於商業用途。詳細條款請參閱 LICENSE 與 COMMERCIAL-LICENSE.md。

## ✨ 核心功能

### 🎨 無限畫布

| 特性 | 說明 |
|------|------|
| 自由縮放、平移 | 無限延伸的繪圖空間 |
| 小地圖導航 | 快速定位畫布位置 |
| 畫布對齊網格 | 精確排列節點 |
| 多畫布切換 | 一個項目多個工作區 |
| 適應畫布 | 一鍵歸位所有節點 |
| 選取操作 | 框選、多選、編組、對齊、複製貼上、撤銷重做 |

### 🤖 AI 節點類型

| 節點 | 說明 |
|------|------|
| 📥 源素材節點 | 承載文本、圖片、影片、音訊素材，支援拖拽導入和節點連線 |
| 🖼️ AI 圖像生成節點 | 輸入提示詞，一鍵生成圖片，支援批量出圖（Banana Pro、GRSAI 等） |
| ✍️ AI 文本生成節點 | 多輪對話、流式輸出，支援 @ 引用其他節點結果（Gemini、GPT 系列等） |
| 🎬 AI 影片生成節點 | 文生影片 / 圖生影片 |
| 🔊 AI 音訊生成節點 | 文字轉語音生成（TTS 模型） |
| ✂️ 剪輯節點 | 把圖片 / 影片片段編排成簡單時間線並匯出 |
| 🧩 拼圖 / 分鏡節點 | 拼接圖片、整理分鏡、生成分鏡腳本 |
| 🌐 瀏覽器節點 | 從網頁提取圖片、影片和文字，發送到畫布繼續創作 |
| 🌐 360 全景圖節點 | 生成沉浸式 360 度全景圖像 |
| 📐 3D 導演台節點 | 建立和編輯 3D 場景，支持模型導入與場景佈局 |
| 💬 注釋節點 | 添加文本注釋、說明和標記 |

### 🛠️ 媒體工具鏈

**圖片處理**：裁剪、標注、遮罩編輯、擦除/重繪、主體識別、扣圖、高清、擴圖、宮格裁剪、360 全景圖。

**影片處理**：裁剪、扣像、擦除、高清、倒放、音畫分離、對口型、抽幀、場景檢測、繼續生成影片。

**音訊處理**：播放預覽、裁剪、音色克隆、音色轉換、人聲分離、語音工作室批量處理。

**任務管理**：任務進度、後台完成提示、聲音提醒、取消、刷新後恢復、結果回填到節點。

### 💾 項目管理

- 多項目切換（左側邊欄項目面板）
- `Ctrl+S` 保存畫布到本地 `user/Canvas Project/` 目錄
- 自動緩存畫布狀態，刷新頁面即時恢復
- 支持資產庫、工作流複用、文件管理、項目資源收集和项目包遷移

### 🧠 創作輔助

- **@ 引用**：在生成節點的提示詞中輸入 `@`，可引用畫布上的節點、資產或上游結果
- **/ 預設**：輸入 `/` 打開預設命令菜單，按當前節點類型快速插入常用提示詞模板
- **Agent 入口**：右下角支援用自然語言描述畫布操作
- **API 連接測試、訂閱中心、VIP 工作流提示和自動更新**

## 🚀 快速開始

1. **下載**：前往 [GitHub Releases](https://github.com/ashuoAI/AI-CanvasPro/releases) 下載最新版（支援 Windows / macOS）
2. **安裝並啟動**：Windows 雙擊 `.exe`，macOS 打開 `.dmg` 或 `.zip`
3. **配置 API Key**：左下角頭像 → 設置 → API 輸入 → 填寫提供商 Key → 保存
4. **開始創作**：雙擊畫布即可快速添加 AI 生成節點
